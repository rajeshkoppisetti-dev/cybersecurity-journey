# Linux Fundamentals - TryHackMe

## Objective
Learn basic Linux commands and how to use them in a terminal.

---

## Commands I Practiced

### 1. pwd
Print current directory

```bash
pwd
```

Output:
```bash
/home/tryhackme
```

---

### 2. ls
List files in directory

```bash
ls
```

Output:
```bash
access.log folder1 folder2 folder3 folder4
```

---

### 3. cd
Change directory

```bash
cd /home
```

Output:
```bash
tryhackme@linux1:/home$
```

---

### 4. cat
Read file content

```bash
cat note.txt
```

Output:
```bash
Hello World !
```

---

## What I Did
- Navigated between directories  
- Viewed files using terminal  
- Practiced basic Linux commands  

---

## What I Learned
The Linux command line is essential in cybersecurity because most tools and servers operate on Linux systems.
