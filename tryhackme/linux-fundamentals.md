## Commands I Learned

cat(concatenate) → read file  
pwd → shows current folder you are in

## What I Did

- Viewed files

## Commands I Practiced

### 1. pwd
Print current directory
Example:
pwd
Output:
/home/tryhackme

---

### 2. ls
List files in directory

Example:
ls
Output:
access.log  folder1  folder2  folder3  folder4

---

### 3. cd
Change directory

Example:
cd /home
Output:
tryhackme@linux1:/home$ 

---

### 4. cat
Read file content

Example:
cat note.txt
Output:
Hello World !

---
